# Knowledge Base Skill

自然语言管理 Zettelkasten 知识库的 Skill。

## 功能

- **知识库管理**：列出、切换、创建知识库
- **快速记录**：用自然语言添加笔记
- **模板创建**：使用预定义模板创建结构化笔记
- **搜索查找**：语义搜索和关键词搜索
- **状态查看**：查看知识库统计信息

## 安装

将本目录复制到 Kimi Code CLI 的 skills 目录：

```bash
# 复制到 Kimi skills 目录
cp -r skill/ ~/.config/agents/skills/knowledge-base

# 或复制到 Claude Code skills 目录
cp -r skill/ ~/.claude/skills/knowledge-base
```

## 使用示例

### 知识库管理

```
有哪些知识库？
切换到工作知识库
创建一个新知识库叫 study
```

### 记录笔记

```
记录一个想法：AI 发展很快
帮我记个笔记，标题是周会记录，内容是讨论了 Q2 计划
用 meeting 模板创建一个产品评审会议记录
```

### 搜索查找

```
搜索 Python 相关的笔记
找一下关于机器学习的笔记
在工作库中搜索项目文档
```

### 查看状态

```
查看知识库状态
列出最近的 10 条笔记
有哪些模板可用？
```

## 依赖

需要安装 `zk-cli` 工具并初始化知识库：

```bash
pip install -e zk-cli/
zk init
```

## 触发词

- "记录..."
- "添加笔记..."
- "搜索..."
- "切换到...知识库"
- "有哪些知识库"
- "用...模板"
- "查看知识库状态"

## 注意事项

1. 始终先确定目标知识库，再执行操作
2. 自动维护会话级知识库上下文
3. 支持连续操作（"再记录一个"）
